# 1. TaskManagementApi
API to create, read, update and delete tasks. Users can categorize tasks, set deadlines, mark as complete, and add notes.

## 1.1 Technologies
- C# (.NET 8)
- MySQL

## 1.2 Frameworks
- Entity Framework
- Pomelo Entity Framework
- AutoMapper
